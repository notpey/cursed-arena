# Cursed Arena ⚔️

A tactical turn-based battle arena game featuring anime-inspired characters with unique abilities and passives.

## Getting Started
```bash
npm install
npm run dev
```

## Recent UI/UX Improvements ✨

The game has received a comprehensive UI/UX overhaul focused on professionalism and user experience:

- **Enhanced Login/Signup Screens**: Modern card-based design with smooth animations and better visual hierarchy
- **Improved Navigation Bar**: Cleaner layout with better spacing, backdrop blur effects, and hover states
- **Better User Profile Display**: Enhanced avatar presentation with gradient backgrounds

### Team Selection
- **Modernized Character Cards**: Improved hover effects, rarity-based styling with glows, and smooth transitions
- **Better Team Preview Slots**: Enhanced drag-and-drop visuals with border animations and scale effects
- **Improved Character Details Panel**: Better organized information with clearer visual hierarchy

### Battle Screen
- **Polished Ability Cards**: Larger, more interactive cards with improved hover states and selection feedback
- **Enhanced Status Effects**: Better badge design with gradients, shadows, and pulsing animations for burning
